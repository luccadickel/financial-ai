# FinancialAI 

Assistente financeiro inteligente com IA, permitindo controle de gastos, receitas e análises por meio de interface web e chat conversacional.

O projeto combina **Spring Boot, JWT, MCP e OpenAI** para oferecer uma experiência híbrida: lançamentos manuais + lançamentos via IA.

---

## Funcionalidades

- ✅ Autenticação com JWT
- ✅ Cadastro e gerenciamento de transações
- ✅ Categorização de gastos e receitas
- ✅ Dashboard com resumos financeiros
- ✅ Chat com IA para registro automático de lançamentos
- ✅ Integração via MCP (Model Context Protocol)

---

## Tecnologias

### Backend
- Java 21
- Spring Boot 3
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Spring AI
- MCP Server / Client
- OpenAI API

### Frontend (em desenvolvimento)
